# createProject

## Version

1.1.0

## Usage

```
var build = require("./createProject/index.js");

build.init()
```

## Api

```
build.init({
    pathroot: './'
})
```

## Use
- step1
    + Enter `node index.js` on the command line
- step2
    + Enter your project name 
- step3
    + Chose project structor according to Tips
- step4
    + Enter your project && use `npm install` to install dependence
    + run
